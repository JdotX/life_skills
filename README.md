解释一下我为什么要做这个skills

现代社会人最稀缺的资源其实是脑细胞。尤其是在一些日常工作中，思考/编排/拉清单其实相当占有一个人的精力。

比如我家请了做饭阿姨，但是我依然对每天思考要不要买菜、以及给阿姨解释某道菜我希望她做的方式上感到头疼，因此我要用AI来让我只需要做添加到收藏夹的动作，并且每周末运行一下skill
